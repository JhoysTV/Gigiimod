# GIGIIMOD Website

Responsive fashion brand website created for **GIGIIMOD**, designed as a visual catalog and direct sales channel.

## Overview

This project presents GIGIIMOD as a contemporary artisan fashion label through a polished editorial-style website. The experience combines:

- A curated product catalog
- A mobile-first responsive layout
- An enriched visual lookbook
- Direct quote and order contact through Instagram and WhatsApp
- A founder section introducing **Genesis Vasquez**, CEO and Fashion Designer of the brand

The main goal of the site is to help visitors discover the collection, connect with the brand, and request product information quickly and naturally.

## Features

- Fully responsive interface optimized for mobile users
- Product filtering and search inside the catalog
- Editorial brand storytelling and lookbook section
- WhatsApp and Instagram contact integration
- Order/cart flow that prepares a ready-to-send customer message
- Founder profile section for brand credibility and personal storytelling

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript

## Project Structure

```text
.
├── Catalogo/      # Product images used in the catalog
├── imagen/        # Branding, editorial visuals, founder photo, and extra assets
├── index.html     # Main page structure
├── styles.css     # Layout, visual system, and responsive behavior
└── script.js      # Catalog filtering, cart logic, and contact message flow
```

## How to Run

Since this is a static website, you can open `index.html` directly in the browser.

For a smoother local preview, you can also serve the project with any static server.

## Notes

- Product and branding assets are stored locally inside the project folders.
- The site is built around direct customer communication instead of a traditional checkout flow.
- WhatsApp and Instagram are both available as quote/contact channels.
